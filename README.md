# 📋 Template

**Starter template for new Workshop-DIY apps** — boilerplate HTML with i18n, themes and standard layout.

## Quick Start

1. Clone this repo
2. Rename the folder to your app name
3. Edit `index.html`, `script.js` and `style.css`

## Files

| File | Description |
|------|-------------|
| `index.html` | Main HTML with i18n and theme support |
| `script.js` | App logic boilerplate |
| `style.css` | Standard Workshop-DIY styles |
| `icon.svg` | Default app icon |
| `logo.svg` | Default logo |
| `*.png` | PWA icons (512px, apple-touch) |
| `favicon.ico` | Favicon |

## Includes

- Trilingual support (EN/FR/AR) with RTL
- Light/dark theme toggle
- PWA-ready icons and manifest structure
- Workshop-DIY branding and layout
